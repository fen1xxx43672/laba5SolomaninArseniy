# Обход бинарного дерева

## 1. Постановка задачи
Необходимо реализовать структуру бинарного дерева и функции для его анализа:
* Три вида обхода в глубину (**DFS**).
* Обход в ширину по уровням (**BFS**).
* Алгоритм расчета суммы значений всех путей от корня к листьям.

## 2. Реализация

### Структура данных
Использован класс `TreeNode`, где каждый узел хранит значение `val` и ссылки на `left` и `right` потомков.

### Методы обхода
1.  **Preorder, Inorder, Postorder**: Реализованы через рекурсию. Это база для работы с деревьями.
2.  **Level-order**: Реализован через итератор и очередь (`deque`). Проходим дерево слой за слоем, что удобно для визуализации структуры.

### Расчет суммы путей
Функция `calculate_all_paths_sum` работает рекурсивно. Она пробрасывает накопленную сумму от родителя к ребенку. Когда алгоритм доходит до "листа" (узла, у которого нет детей), он возвращает итоговую сумму этого конкретного пути. Все полученные значения суммируются.

## 3. Пример дерева для тестов
В коде построено следующее дерево:
* Корень: **10**
* Левое плечо: **5**, у которого есть дети **2** и **7**
* Правое плечо: **15**, у которого есть ребенок **20**



## 4. Результаты работы программы

При запуске на тестовых данных получены следующие результаты:

| Метод обхода | Результат |
| :--- | :--- |
| **Preorder** | `[10, 5, 2, 7, 15, 20]` |
| **Inorder** | `[2, 5, 7, 10, 15, 20]` |
| **Postorder** | `[2, 7, 5, 20, 15, 10]` |
| **Level-order** | `[[10], [5, 15], [2, 7, 20]]` |

**Итоговая сумма всех путей:** `84`
*Логика расчета:*
1. Путь 1: $10 + 5 + 2 = 17$
2. Путь 2: $10 + 5 + 7 = 22$
3. Путь 3: $10 + 15 + 20 = 45$
**Сумма:** $17 + 22 + 45 = 84$
